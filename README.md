MindMatch is a browser-based Memory Card Game built using pure HTML, CSS, and JavaScript.  
The game challenges players to match pairs of cards while tracking time and moves.
---

🎯 Project Objective

The objective of this project is to develop an interactive memory-based card matching game using only frontend technologies.  
It demonstrates DOM manipulation, event handling, game logic implementation, animations, and state management without using any frameworks or libraries.

---

🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Local Browser Storage
- Audio Integration

---

✨ Features
- 🎴 Card flip animation
- ⏱️ Real-time timer
- ⭐ Move counter
- 🔄 Restart game button
- 🔊 Sound effects
- 🎉 Winning popup message
- 📱 Responsive layout

---

🧠 How to Play
1. Click on any card to flip it.
2. Flip two cards at a time.
3. If the cards match, they stay open.
4. If not, they flip back.
5. Match all pairs to win the game.
6. Try to complete in minimum time and moves.
